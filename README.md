# Proyecto de Tecnología Musical con Software Libre

## Descripción

Este proyecto utiliza una grabación de campo propia realizada en un entorno rural, con sonidos de gallinas, gallos y patitos.

La propuesta explora el uso de herramientas de software libre para la creación y manipulación sonora.

## Herramientas utilizadas

* Pure Data
* SuperCollider

## Comunicación

Los programas se conectan mediante el protocolo OSC (Open Sound Control).
